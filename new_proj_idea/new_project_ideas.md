## 💡 신규 프로젝트 제안

### 🅰️ **내부 문서 검색 엔진 (DocSearch)**
> 변환된 문서들을 즉시 검색 가능하게 하는 로컬 검색 시스템

- **문제**: DocCon으로 변환한 수백 개 문서에서 원하는 정보를 빠르게 찾기 어려움
- **솔루션**: 
  - 변환된 `.md` 파일들을 인덱싱하여 전문(全文) 검색
  - 폴더/날짜/파일유형별 필터링
  - 검색 결과 미리보기 + 원본 파일 바로 열기
  - 키워드 하이라이팅 및 관련 문서 추천
- **기술**: Python + Flask/FastAPI + SQLite FTS5 (or Whoosh) + Modern Web UI
- **연계**: DocCon 변환 출력 폴더를 감시해서 자동 인덱싱

### 🅱️ **업무 문서 자동 요약/분석 대시보드 (DocInsight)**
> AI 기반으로 문서를 분석하고 핵심 인사이트를 추출하는 대시보드

- **문제**: 대량 문서의 핵심 내용을 빠르게 파악하기 어려움
- **솔루션**:
  - 문서별 자동 요약 (AI 활용)
  - 핵심 키워드/엔티티 추출
  - 문서 간 관계도 시각화
  - 시간별 문서 트렌드 분석
- **기술**: Python + LLM API + D3.js 시각화 + Streamlit
- **연계**: NotebookLM 파이프라인과 연동

### 🅲️ **병원 업무 자동화 허브 (WorkflowHub)**
> 반복적인 병원 행정 업무를 자동화하는 통합 관리 도구

- **문제**: 반복적인 문서 작성, 보고서 생성, 데이터 집계 등에 시간 소요
- **솔루션**:
  - 템플릿 기반 문서 자동 생성 (임명장, 공문 등)
  - Excel 데이터 자동 집계 + 보고서 생성
  - 정기 업무 스케줄링 및 알림
  - 업무 진행 상황 트래킹 대시보드
- **기술**: Python + PowerShell + WinForms/Electron + Task Scheduler
- **연계**: DocCon + 시나리오맵 기능 통합

### 🅳️ **문서 변환 모니터링 & 통계 Web Dashboard (DocCon Web)**
> DocCon의 변환 이력을 실시간 모니터링하고 통계를 제공하는 웹 대시보드

- **문제**: DocCon GUI로는 과거 변환 이력과 통계를 보기 어려움
- **솔루션**:
  - 변환 이력 DB 저장 + 시계열 통계
  - 성공/실패 비율, 파일 유형별 분석
  - 실시간 변환 진행 상황 WebSocket 모니터링
  - 원격에서도 변환 작업 시작/관리
- **기술**: Python + FastAPI + SQLite + Vanilla JS + Chart.js
- **연계**: DocCon 엔진을 API로 래핑

### 🅴️ **Smart File Organizer (SmartSort)**
> AI 기반 파일 자동 분류 + 정리 도구

- **문제**: 공유 드라이브에 정리되지 않은 파일이 산재
- **솔루션**:
  - 파일 내용 기반 자동 카테고리 분류
  - 중복 파일 탐지 + 정리 제안
  - 자동 명명 규칙 적용
  - 폴더 구조 최적화 제안
- **기술**: Python + LLM + GUI (WinForms or Web)
- **연계**: DocCon 중복제거 + HWP Parser 기술 재활용
